ls - показывает все файлы/директории(папке) в текущей директории
ls test - показывает все файлы и папки внутри указанной директории "test"
ls -a - показывает все файлы внутри директории, а так же все скрытык файлы/директории
ls -l - показывает более подробную информацию о файлах и папках
ls -la - показывает более подробную информацию еще о скрытых файлах/папках

cd - переходит в домашнюю папку
cd "test" - переходит в заданную директорию (в нашем случае в test)
cd .. - переходит на одну  директорию назад
cd ../.. - переходит на 2 директории назад (чтоьы перейти еще дальше можно через / указывать по 2 точки)

mkdir test - создает директорию test (именно в текущей директории)
mkdir test1/test2 - создает папку test2 в папке test1 (если директория не существует, то выйдет ошибка)
mkdir test1 test2 test3 test4 - создает 4 директории (test1 - test4) именно в текущей папке

touch test.txt - создает файл с любым расширением в текущей директории
touch test1.py test2.py test2.txt - создает много файлов с разными расширениями одновременно

rm test.txt - удаляет файл test.txt, если такого файла нет то он выкинет ошибку, если укзать директорию - так-же выкинет ошибку
rmdir - удаляет директорию, удаляет только пустую, если нет файлов и папок
rm -rf test - удаляет файл или папку совсем содержимым (удаляет безвозвратно)

mv test.txt test1/ - перемещает файл test.txt в директорию test
mv test.txt test2.py - переименрвывает файл test.txt в test2.py
mv test/ lesson1/ 

history - показывает все команды, которые были выполнены в этом терминале
history 10 - показывает 10 последних команд
history | grep cd - показывает все команды с использованием слова cd
history 1 | grep 'cd' - для mac os 

pwd - показывает полный путь начиная с корневой папки и до текущей директории

nano test.txt - открывает файл test.txt в редакторе кода nano (если такого файла нет, то создает его)
    чтобы выйти из nano:
        1) ctrl + x 
        2) y 
        3) ENTER

cat test.txt - выводит в терминал все содержимое файла

code . - открывает редактор кода vs code  в текущей директории

sudo - команда которая дает права суперюзера (super user do)
